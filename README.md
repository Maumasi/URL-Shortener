# URL-Shortener

###### simplified URL shortener application

## Package dependencies

Install all dependencies.

```
npm i

```

## Endpoints

### URL

POST localhost:3000/api/v1/url - Create shortened link

```json
{
  "URL": "http://bs.com/66vo7fl7ar8"
}
```
